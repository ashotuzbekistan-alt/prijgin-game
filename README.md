# Прыжкин: Приключение

Платформер в духе классических денди-игр — 30 уровней + 6 боссов (каждый
5-й уровень), написан на Python + Pygame. Работает на компьютере (EXE) и
на телефоне (Android APK, собирается автоматически на GitHub).

Все 30 уровней сгенерированы и **проверены честной физической симуляцией**
(`simulate_levels.py`) — гарантированно проходимы, без дыр-ловушек и
непреодолимых участков.

## Управление

**На компьютере:**
- Стрелки / A,D — движение
- Пробел / W / Стрелка вверх — прыжок

**На телефоне (Android):**
- Виртуальный джойстик слева внизу — движение
- Кнопка «ПРЫЖОК» справа внизу — прыжок

Общее для всех платформ: прыжок на голову врага — убивает его. Каждый
5-й уровень (5, 10, 15, 20, 25, 30) — бой с боссом, который становится
крепче с каждым разом. Собирай монетки, доберись до флага/победи босса —
переход на следующий уровень.

## Как выложить проект на GitHub

1. Зайди на [github.com](https://github.com), нажми **New repository**
2. Дай название (например `prygkin-game`), выбери **Public**, создай
3. На своём компьютере, в папке проекта, выполни по очереди:
   ```
   git init
   git add .
   git commit -m "Первая версия игры"
   git branch -M main
   git remote add origin https://github.com/ТВОЙ_НИКНЕЙМ/prygkin-game.git
   git push -u origin main
   ```
   (если `git` не установлен — скачай с [git-scm.com](https://git-scm.com))
4. Обнови страницу репозитория на GitHub — весь код уже там

## Как получить готовый APK (сборка происходит в облаке, бесплатно)

Сборка Android настроена через **GitHub Actions** — тебе для этого не
нужен Linux и вообще ничего устанавливать локально, всё соберётся на
серверах GitHub автоматически.

1. После того как запушил код (см. выше), зайди на вкладку **Actions** в
   своём репозитории на GitHub
2. Увидишь запущенный workflow **"Build Android APK"** — подожди
   10-20 минут (первая сборка самая долгая)
3. Когда сборка завершится (зелёная галочка), открой её и внизу страницы
   найди раздел **Artifacts** → скачай `prygkin-apk` (это zip с APK внутри)
4. Распакуй zip — внутри будет сам `.apk` файл

## Как раздать APK знакомым

Проще всего — через **Release**:
1. На странице репозитория справа найди **Releases** → **Create a new release**
2. Придумай тег версии, например `v1.0`, опубликуй
3. Через пару минут после публикации тега сборка сама прикрепит готовый
   `.apk` прямо к этому Release (это тоже настроено в workflow)
4. Скидываешь знакомым просто ссылку на Release — они скачивают `.apk`
   прямо со страницы GitHub

На телефоне получателя: скачать файл → открыть → разрешить "установка
из неизвестных источников" (Android спросит сам, один тап) → установить
→ играть.

## Локальный запуск на компьютере (для проверки перед публикацией)

```
pip install -r requirements.txt
python main.py
```

## Сборка в EXE (Windows)

Проще всего — двойным кликом запустить `build_exe.bat`. Либо вручную:
```
pip install pyinstaller
pyinstaller --onefile --windowed --name "Prygkin" main.py
```
Готовый `.exe` появится в папке `dist`.

## Структура проекта

```
main.py             — точка входа
game.py             — игровой цикл, меню, HUD, сенсорное управление
entities.py         — игрок, враги, босс, блоки, монеты, снаряды
world.py            — построение уровня из текстовой карты
levels.py           — 30 готовых уровней (сгенерированы, см. ниже)
level_generator.py  — генератор уровней (можно перезапустить для новых карт)
simulate_levels.py  — честная проверка проходимости всех уровней
settings.py         — константы и настройки
buildozer.spec      — конфигурация сборки под Android
.github/workflows/  — автоматическая сборка APK на GitHub
```

## Как перегенерировать уровни (если захочешь другие)

```
python level_generator.py > levels.py
python simulate_levels.py
```
Второй командой обязательно проверь, что вывод заканчивается строкой
**"ВСЕ УРОВНИ ПРОХОДИМЫ"** — если нет, не используй эти уровни, перезапусти
генератор (можно поменять `random.Random(2026)` на другое число в
`level_generator.py`, чтобы получить другой набор уровней) и проверь снова.
